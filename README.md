# Blue Marble Next Generation UDIM Textures

This repo acts as a backup to NASAs Blue Marble textures.
See [link](https://science.nasa.gov/earth/earth-observatory/blue-marble-next-generation/)

All files were created as part of the BMNG Project by NASA Earth Observatory.
