# NASA Earth UDIM Textures

This repo acts as a backup to NASAs "Blue Marble" and "Black Marble" textures:
- [Blue Marble Next Generation](https://science.nasa.gov/earth/earth-observatory/blue-marble-next-generation/)
- [Black Marble](https://www.earthdata.nasa.gov/data/projects/black-marble)
